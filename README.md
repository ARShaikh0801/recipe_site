# Recipe Site

A static recipe website.

## Overview
A simple site to display recipes with ingredients and instructions, written with HTML and styled with CSS.

## Features
- Recipe listing and detail pages
- Clean layout for steps and ingredients

## Tech stack
- HTML, CSS

## Running locally
1. Clone the repo:
   git clone https://github.com/ARShaikh0801/recipe_site.git
2. Open `index.html` in your browser.
